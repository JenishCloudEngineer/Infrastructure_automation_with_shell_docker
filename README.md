# Infrastructure_automation_with_shell_docker

cd Infrastructure_automation_with_shell_docker

change with your credential inside .env.example file and save it as .env

run 
   chmod +x setup.sh
   ./setup.sh

It will create all the services from docker compose

## URL to visit

App Health Check	http://localhost/health

App Logs	http://localhost/logs

Redis Status	http://localhost/redis-status

Redis Info	http://localhost/redis-info

Main App	http://localhost

Jenkins	http://localhost:9090
